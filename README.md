# YouTube Clone – Frontend

This is the frontend of the YouTube Clone, a web-based video streaming platform inspired by YouTube’s modern interface and user experience.
Built with React, the frontend delivers a smooth, dynamic, and responsive UI for browsing, searching, and watching videos.

The project integrates with a backend API for video data, comments, authentication, and user interactions.

---

### Purpose

The purpose of this project is to recreate the essential functionalities of YouTube while providing clean code and a scalable frontend architecture.
The platform allows users to explore trending videos, search for content, watch videos, and navigate between channels in a seamless and responsive digital environment.

---

### Tech Stack

· React.js
· CSS Modules/ TailwindCSS
· Context API
· React Router DOM
· axios 
· React Player
